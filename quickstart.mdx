## 1\. Create an account and get your API key

[Create an account](https://searchlysis.com/key)

Get your [API key](https://searchlysis.com/key).

## 2\. Make an API request

### Base URL

```txt
https://searchlysis.com/api
```

### Authentication

```json
{
  "headers": {
    "x-api-key": "sk_123456789"
  }
}
```

### Endpoints

#### `/v1/search`

```sh
curl --location 'https://searchlysis.com/api/v1/search' \
--header 'Content-Type: application/json' \
--header 'x-api-key: sk_123456789' \
--data '{
    "query": "Searchlysis",
    "prompt": "Summarize it"
}'
```

#### `/v1/links`

```sh
curl --location 'https://searchlysis.com/api/v1/links' \
--header 'Content-Type: application/json' \
--header 'x-api-key: sk_123456789' \
--data '{
    "link": "https://searchlysis.com",
    "prompt": "Summarize it"
}'
```

#### `/v1/analyses`

```sh
curl --location 'https://searchlysis.com/api/v1/analyses?id=analysis_1234567' \
--header 'x-api-key: sk_123456789'
```

#### `/v1/analyses/list`

```sh
curl --location 'https://searchlysis.com/api/v1/analyses/list?page_num=1&page_size=10' \
--header 'x-api-key: sk_123456789'
```
