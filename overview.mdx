## What is Searchlysis

Searchlysis can analyze any web content and has two core features:

### [/Analyze Search ->](/api-reference/endpoint/analyze-search)

Use Searchlysis to analyze web content with LLMs based on SERPs.

### [/Analyze Links ->](/api-reference/endpoint/analyze-links)

Use Searchlysis to analyze web content based on direct links with LLMs.

## Quickstart

The first step to start using Searchlysis is to [create an account](https://searchlysis.com/key) and get your [API key](https://searchlysis.com/key).

After that, feel free to explore the [API reference](/api-reference) for more details. Or to jump into the [Quickstart](/quickstart).
