## What is Searchlysis

Searchlysis can analyze any web content and has two core features:

<CardGroup cols="2">
  <Card title="Analyze Search" icon="sparkles" href="/api-reference/endpoint/analyze-search">
    Use Searchlysis to analyze web content with LLMs based on SERPs.
  </Card>
  <Card title="Analyze Search" icon="sparkles" href="/api-reference/endpoint/analyze-links">
    Use Searchlysis to analyze web content based on direct links with LLMs.
  </Card>
</CardGroup>
